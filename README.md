MCP: Multi-Client Proxy for Automation
MCP (Multi-Client Proxy) is not a Minecraft tool, but a protocol for LLMs and AI agents.  It allows aggregation of multiple backend services (e.g., AI tools, APIs) into a unified interface using SSE or HTTP streaming.  It supports session isolation, request routing, and failover mechanisms.  Tools like FastMCP and TBXark/mcp-proxy enable proxying LLM clients (e.g., Gemini, ChatGPT) and are used in AI workflows, not Minecraft server management.
